Recent Projects
 
Human Activity Recognition using Ensemble Convolutional Neural Networks

This project involved the development of a machine learning model to recognize an array of human activities from timeseries sensor data obtained from mobile phones. Common, daily physical activities such as walking, jogging, climbing, sitting etc. were included. The model used was an ensemble network of multiple convolutional neural networks. This approach was used in order to combine the best features of multiple models and in order to forego the heuristic development of features as with a more traditional model. The project resulted in a research publication which can be found here: https://ieeexplore.ieee.org/document/9400290

Google Tag Manager Data Logger

One of the challenges encountered at work was debugging errors enocuntered in server-side Google Tag Manager (GTM). GTM is a tool that allows the management of tags or snippets of code used in websites for marketing purposes. Sometimes, certain tags would not register and it was often difficult for analysts to decipher the errors without any means of viewing the particulars of the requests which raised the errors. The solution implemented involved setting up a serverless Cloud Function that parses particular requests with an extra token in the header. This token was compared against an existing one and subsequently, the request data was logged into a Google Cloud Storage bucket. The technologies involved in this project included JavaScript, Python, Google BigQuery and Google Cloud Functions.
